#主题
云端只有两个主题分别是上行和下行
上行主题 /uplink
终端设备通过该主题发布数据给到平台侧

下行主题 /downlink
云端通过该主题发布数据给到终端(目前未用到)


#数据格式
上行数据格式按照json的规范。

##通用格式
{
	"cmd":100,
	"id":xxx,
	"payload":
	{
	}	
}
说明：cmd 为命令码，开始为100
id：id为命令序列号从0开始到65535，往复循环
payload:为该命令下携带的有效数据，应根据以下命令说明来填充相应的数据


###上报数据命令码100
{
	"cmd":100,
	"id":xxx,
	"payload":
	{
		"temp":21,
		"humi":70,
		"lon":3743.519728
	}	
}

cmd =100表示上报数据的命令码
payload的参数说明如下
temp为上报的终端设备的温度信息
humi为上报的终端设备的温度信息
lon为上报的终端设备的gps信息
